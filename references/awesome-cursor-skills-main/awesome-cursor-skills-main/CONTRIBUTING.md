# Contributing

Thank you for contributing to Awesome Cursor Skills!

## Guidelines

- Search existing entries before adding a new one.
- Use the following format: `- [Name](link) - Short description ending with a period.`
- Add new entries to the bottom of the relevant category.
- If no category fits, suggest a new one in your PR description.
- Check that your links are working.
- One pull request per addition.

## Quality Standards

Entries should:
- Be actively maintained (updated within the last year).
- Have clear documentation or a `SKILL.md` file.
- Be compatible with Cursor (`.cursor/skills/` or plugin format).
- Provide genuine value — not just a wrapper around a single prompt.

## Adding a New Category

If you think a new category is needed, include it in your PR with at least 3 entries.
