| Alan | Detay |
|------|-------|
| **Duyuru Tarihi** | 5 Mart 2026 |

| **Statü** | GA (Genel Kullanım) |

Cursor Automations, IDE tabanlı ajanların bilgisayar başında değilken bile çalışmasını sağlayan devrimsel bir özelliktir. Ajanlar artık sadece "prompt-and-monitor" mantığıyla değil, **tetikleyicilerle (triggers)** otonom olarak çalışmaktadır.

**Kullanım Örnekleri:**
- GitHub'da bir PR açıldığında otomatik kod incelemesi
- Slack mesajından JIRA task oluşturup kodu yazma
- PagerDuty uyarısında logları okuyup hata düzeltme PR'ı açma

**Nasıl Kullanılır:**

Otomasyonlar Cursor Dashboard üzerinden yapılandırılır:

1. **Tetikleyici (Trigger) Seçin:** GitHub PR (Push), Slack Webhook, Linear Issue, PagerDuty Incident veya Cron (Zamanlanmış Görev)
2. **Görev (Task):** Ajanın ne yapması gerektiğini `.cursorrules` veya özel bir komutla belirleyin

**Örnek görev tanımı:**
```
Run apidog test. If fails, create Linear ticket and ping #dev-alerts on Slack.
```

## 2. 🐛 Bug Raporları ve Workaround'lar

### Zombi Geri Alma Hatası (The "Zombie Revert" Bug) — GÜNCELLENDİ

| Alan | Detay |
|------|-------|
| **Ciddiyet** | Critical 🔴 |
| **İlk Rapor** | Ocak 2026 |
| **Son Güncelleme** | Mart 2026 |
| **Etkilenen Versiyon** | 2.4.x - 2.6.x |
| **Durum** | 🔵 Workaround Available / Kısmen Düzeltildi |

Cursor ekibi bu ay hatanın temel nedenlerini doğruladı:

**1. Agent Review Çakışması:**
Aynı dosyanın hem standart editörde hem de "Agent Review" sekmesinde açık olması, "Fix in Chat" denildiğinde değişiklikleri anında eziyor.
- ✅ **Çözüm:** Prompt göndermeden önce Agent Review sekmesini kapatın.

**2. Cloud Sync Çakışması:**
OneDrive, Google Drive veya iCloud'un Cursor'ın dosya kilitleme mekanizmasından daha hızlı senkronize olmaya çalışması dosyayı eski versiyona döndürüyor.
- ✅ **Çözüm:** Proje klasörünüzü bulut senkronizasyonundan çıkarın.

**3. Format On Save Çakışması:**
Prettier, Black, ESLint gibi uzantıların Format on Save ile aktif olması AI'ın yazdığı blokları eziyor.
- ✅ **Çözüm:** AI ile yoğun çalışırken `cursor --disable-extensions` kullanın veya Format on Save'i kapatın.

**Kaynak:** [Cursor Forum - AI Code Changes Revert](https://forum.cursor.com/t/ai-code-changes-immediately-revert-on-specific-files/148878/3)

### Linux (Ubuntu/GNOME) Çökme Döngüsü

| Alan | Detay |
|------|-------|
| **Ciddiyet** | High 🟠 |
| **İlk Rapor** | 10 Mart 2026 |
| **Etkilenen Versiyon** | 2.6.x |
| **Platform** | Linux (Ubuntu 24.04 Wayland/GNOME) |
| **Durum** | 🔵 Workaround Available |

Ubuntu 24.04 GNOME/Wayland kullanıcılarında, şifre girilip oturum açıldıktan sonra tüm masaüstü oturumunun çöküp login ekranına dönmesi.

**Workaround:**
```bash
# Terminale tty üzerinden giriş yapın veya kurtarma modunu kullanın
rm -rf ~/.config/Cursor
```

Bozuk Cursor config/cache dosyası GNOME'u çökertmektedir. Klasörü silmek temiz bir başlatma sağlar.

**Kaynak:** [Reddit - Fix GNOME crashing due to config folder](https://www.reddit.com/r/cursor/comments/1rpabrf/how_to_fix_gnome_crashing_due_to_config_folder/)

## 6. 🔥 Yeni Workflow'lar

### 1. Context 7 MCP Server (Oyun Değiştirici)

2000'den fazla programlama kütüphanesinin en güncel dokümantasyonunu anlık olarak LLM'e çeker. Ajanın eski versiyon kodu (halüsinasyon) yazmasını engeller.

**Kurulum (`.cursor/mcp.json`):**
```json
{
  "mcpServers": {
    "context7": {
      "command": "npx",
      "args": ["-y", "@upstash/context7-mcp"]
    }
  }
}
```

### 2. "Linear vs Looped" İş Akışı

Eski nesil "Vibe Coding" → out. "Agentic Engineering" → in. (Andrej Karpathy)

- **Linear (Doğrusal):** Hızlı prototip veya basit bug fix → `Cmd+K` ile "Prompt → Kod"
- **Looped (Döngüsel):** Karmaşık özellikler → Ajanı "Planla → Test Yaz → Kodla → Test Et → Hata Logunu Oku → Düzelt" döngüsüne sokmak

### 3. Yeni Nesil `.cursorrules` İpuçları

**Anti-Halüsinasyon Kuralı:**
```markdown
Eğer bir npm kütüphanesini import edeceksen, önce terminal ajanını kullanarak
`npm list` komutunu çalıştır ve o kütüphanenin projede yüklü olduğundan %100 emin ol.
```

**Interface Freeze Kuralı:**
```markdown
Yeni bir özellik eklerken, ASLA direkt uygulama (implementation) yazma.
Önce types/interfaces tanımla, onayımı bekle, sonra fonksiyonu yaz.
```
